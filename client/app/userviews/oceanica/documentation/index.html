Overview

Oceanica is an incredibly beautiful responsive Bootstrap Template for corporate and creative professionals. It works on all main web browsers, tablets and phones. Thank you for using Oceanica.

  • Author: Pixel Press (formerly Creligent Themes)
  • Contact: hellopixelpress@gmail.com

Main Structure

The theme consists of a single main page: index.html. All the sections in the page are included in this file.

Page Meta Data

The code below is included in the head section. This is used to describe website content and optimize the website for search engines.

<meta charset="utf-8" />
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1, user-scalable=no">
<meta name="description" content="Responsive HTML5 Single Page Design Template">
<meta name="keywords" content="Bootstrap, Bootstrap 3, Font Awesome, Ion Icons, Ionicons, HTML5, CSS, CSS3, JavaScript, Template, JS, jQuery, Ajax, work, single-page, one-page, Theme, Twitter, Facebook">

Pre Loader

Theme includes a JavaScript based pre-loader which appears on the top of the page. Code for the same is shown below.

<!-- PRE-LOADER JAVASCRIPT START -->
<script src="js/pace.min.js" type="text/javascript"></script>
<!-- PRE-LOADER JAVASCRIPT END -->

CSS and Fonts

The CSS and Fonts are included next. style.css is Oceanica specific CSS and can be modified easily to customize the theme.

<!-- STYLESHEETS START -->
<!-- Bootstrap style sheet -->
<link rel="stylesheet" href="css/bootstrap.css" />
<!-- Font Awesome style sheet -->
<link rel="stylesheet" href="css/font-awesome.min.css">
<!-- Ion Icons style sheet -->
<link rel="stylesheet" href="css/ionicons.min.css">
<!-- Magnific popup style sheet -->
<link rel="stylesheet" href="css/magnific-popup.css">
<!-- Oceanica specific style sheet -->
<link rel="stylesheet" href="css/styles.css" />
<!-- STYLESHEETS END -->

<!-- FONTS START -->
<link href="http://fonts.googleapis.com/css?family=Lato:300,400,700" rel="stylesheet" type="text/css">
<!-- FONTS END -->

Navigation

Oceanica uses one main navigation for the website. It is included in the body section.

<!-- MAIN NAV START -->
<nav id="top-navigation" class="navbar navbar-default navbar-transparent navbar-fixed-top" role="navigation">
    <div class="navbar-header">
        <button id="menu-mobile" type="button" class="navbar-toggle" data-toggle="collapse" data-target="#navbar-collapse-1">
            <span class="fa fa-bars"></span>
        </button>
        <!-- LOGO COMES HERE -->
        <a class="navbar-brand scroll" href="#header"><i class="fa fa-sun-o logo-circled"></i> Oceanica</a>
    </div>

    <!-- NAV ITEMS START -->
    <div class="collapse navbar-collapse" id="navbar-collapse-1">
        <ul class="nav navbar-nav navbar-right">

</nav>

Page Sections

The first section of the page is enclosed in header tag.

<!-- HOME START -->
<header id="header">
    <div class="jumbotron">
        <div class="container">
            ... ... ...
        </div>
    </div>
</header>
<!-- HOME END -->

The middle sections are enclosed in section tags, which can be customized as per your content. The sections have appropriate ids which link them to the main navigation.

<!-- ABOUT START -->
<section id="about">
    <div class="container">
        <div class="row">
            ... ... ...
        </div>
    </div>
</section>
<!-- ABOUT END -->
<!-- SERVICES START -->
<section id="services">
    <div class="container">
        <div class="row">
            ... ... ...
        </div>
    </div>
</section>
<!-- SERVICE END -->

The last section of the page is enclosed in footer tag.

<!-- FOOTER START -->
<footer id="footer">
    <div class="container">
        <div class="row">
            <div class="col-xs-12">
                <p class="text-muted">© Copyright. All rights reserved. Made with extreme care by Pixel Press (formerly Creligent Themes).</p>
            </div>
        </div>
    </div>
</footer>
<!-- FOOTER END -->

JavaScript

Javascript files loaded in the end of page. This will reduce page load time.

<!-- JAVASCRIPT START -->
<!-- jQuery script -->
<script src="js/jquery-1.10.2.min.js" type="text/javascript"></script>
<!-- Bootstrap script -->
<script src="js/bootstrap.min.js" type="text/javascript"></script>
<!-- Magnific Popup script -->
<script src="js/jquery.magnific-popup.min.js" type="text/javascript"></script>
<!-- Parallax script -->
<script src="js/jquery.parallax-1.1.3.js" type="text/javascript"></script>

<!-- Oceanica specific Scripts -->
<script src="js/scripts.js" type="text/javascript"></script>
<!-- JAVASCRIPT END -->

PHP

We can help you setup email when someone submits the contact form. For this we support PHPMailer library. If you use this library, just send an email to us and we will provide an upated send_message.php file.

In other cases you can code your own logic to handle contact form.

Fonts

Oceanica uses Lato in 3 weights from Google Fonts.

The font is imported in the head section of index.html

<!-- FONTS START -->
<link href="http://fonts.googleapis.com/css?family=Lato:300,400,700" rel="stylesheet" type="text/css">
<!-- FONTS END -->

Icons

Oceanica supports 3 icon fonts: Font Awesome (v4.0), Ion Icons and Glypicons (provided by Bootstrap). The icon fonts are included in the head section of index.html

<!-- Bootstrap style sheet -->
<link rel="stylesheet" href="css/bootstrap.css" />
<!-- Font Awesome style sheet -->
<link rel="stylesheet" href="css/font-awesome.min.css">
<!-- Ion Icons style sheet -->
<link rel="stylesheet" href="css/ionicons.min.css">
